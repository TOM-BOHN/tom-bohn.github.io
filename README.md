## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Export static site for GitHub Pages
npm run export
```

## Deployment

This site is configured for GitHub Pages. After building, the `out/` directory contains the static files ready for deployment.

## Custom Domain

The `CNAME` file is located in the `public/` directory. This ensures it's included in the static export and deployed to GitHub Pages.
